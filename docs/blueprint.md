# **App Name**: AeroTrack

## Core Features:

- Supply Tracking: Track aircraft maintenance supplies using a database system, integrated with a UI to easily browse and visualize supplies on-hand.
- Tool Management: Manage the tooling inventory using the same database system, tracking tools on-hand, when they were last calibrated, and by whom.
- Stock Reordering Tool: The reordering tool notifies stakeholders when stock runs low and needs to be reordered.
- Supply Usage Reporting: The reporting tool gathers insights on the amount of supply that has been used to show stakeholders what areas are experiencing waste or abnormal demand.
- Operation Recording: Record all operations within the system using a database.
- Image Storage: Store images within the system using a dedicated database.

## Style Guidelines:

- Primary color: Deep sky blue (#3091ff), reminiscent of the open sky.
- Background color: Very light blue (#f2faff) to keep it light.
- Accent color: Orange (#FF8C30) for an energetic flair.
- Body and headline font: 'Inter' for a modern, neutral aesthetic.
